  • Lab 1: Foundations of the Web

    Overview

    The lab has two main sections - Web and HTTP Protocol, and web application programming. In the Web and HTTP Protocol section, we used the Wireshark tool to analyze the HTTP protocol. HTTP is a synchronous protocol that contains two main components - the HTTP request sent by the browser to the server, and the HTTP response sent back by the server. The browser makes a request and waits for the response before continuing..

  • Lab 2: Front-end Web Development

    Overview

    The lab has two main sections. The first section covers basic HTML with forms and JavaScript. Here I created a simple HTML webpage with forms and sent the form data to an Echo.php file. I also used JavaScript to send requests using GET and POST methods. The second section focuses on AJAX, CSS, jQuery and Web API integration. In this section, I used AJAX and jQuery to send form data asynchronously. I utilized CSS for styling and modifying the structure of the HTML content. Overall, the first section dealt with basic HTML form handling while the second section incorporated more advanced techniques like AJAX for asynchronous communication..

  • Hackations
  • Hackathon 1: Cross-site Scripting Attacks and Defenses

    Overview

    The main focus of this hackathon was on implementing XSS (cross-site scripting) attacks and defending against them. While hacking is illegal, it's important for developers to understand these types of attacks in order to properly defend against them. The hackathon was divided into two main tasks: Attacking a website through 6 different levels of XSS attacks. This involved actually carrying out the attacks to practice and see their effects. Defending the HTML and PHP code from previous labs against XSS attacks. This involved identifying vulnerabilities in the code and implementing protections like input validation and output encoding. Though hacking was involved, the overall goal was to gain hands-on experience with XSS in order to better secure systems against such attacks in an ethical manner.
